Introducción a JavaScript JavaScript es un lenguaje de programación que permite agregar interactividad y dinamismo a las páginas web. Mientras HTML estructura el contenido y CSS lo estiliza, JavaScript controla el comportamiento, permitiendo responder a eventos, validar formularios, mostrar mensajes, modificar elementos del DOM y mucho más. JavaScript es se ejecuta directamente en el navegador, sin necesidad de instalar programas adicionales, y es el lenguaje más utilizado en el desarrollo web. Realizar tarea para mañana
